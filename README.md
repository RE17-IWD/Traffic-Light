Traffic Light — Arduino Project
A simple traffic light using 3 LEDs and an Arduino.
What You Need

Arduino Uno
Red, Yellow, Green LED (1 each)
220Ω resistor (3x)
Breadboard + jumper wires

Wiring
LED Arduino Pin Through Red Pin 13 220Ω → GND
YellowPin 12 220Ω → GND
GreenPin 11 220Ω → GND
How to Run

Open Arduino IDE
Paste the code and upload
The lights cycle: Red → Green → Yellow → repeat
